## Getting Started

### Frameworks
Frameworks implemented in this project are
* [Mocha](https://www.npmjs.com/package/mocha)
* [Chai](https://www.npmjs.com/package/chai)
* [SuperTest](https://www.npmjs.com/package/supertest)
